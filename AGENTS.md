# AGENTS.md

## Instrucciones para Codex

Lee primero `CODEX_CONTEXT.md` antes de modificar código.

Este proyecto es un sistema de nómina, vacaciones, préstamos de empleados y desvinculación laboral.

Antes de cambiar archivos:

1. Revisa los modelos relacionados.
2. Revisa los controladores existentes.
3. Revisa helpers ya creados.
4. No dupliques lógica si ya existe.
5. Mantén compatibilidad con registros viejos.
6. Devuelve cambios completos y explica qué archivos tocaste.

Stack:

- Backend: Node.js, Express, TypeScript, Mongoose, MongoDB.
- Frontend: Vue 3 + Quasar.
- Zona horaria: America/Santo_Domingo.

Preferencias:

- Código completo.
- No cambiar nombres de endpoints sin necesidad.
- No romper flujos existentes.
